### Описание:  
#### Задание_1.txt это "Тестовое задание №1"

#### Установить aiohttp
#### Запустить start.py и запустятся все файлы задания.
#### Или можно запускать по 1 файлу
#### Для изменений тестовых данных можно вставить свои значения в каждом файле
